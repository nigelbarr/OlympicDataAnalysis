<h1>R - Olympic Performance and Socioeconomic Analysis</h1>

<h2>Description</h2>
This project investigates how GDP, education, government type, and athlete physical attributes (e.g., height) correlate with Olympic performance using multi-source global datasets.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Rstudio</b> 
- <b>R programming language</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
